# Harvester Installer Documentation

This directory contains documentation for the Harvester Installer project.

## Directory Structure

### `efi/`
Documentation related to EFI/UEFI boot support:
- **ARCHIVE_EFI_INFO.md** - Information about the EFI essentials archive
- **BUILD_STATUS.md** - Build status and monitoring guide
- **DIFF_EFI_CHANGES.md** - Technical documentation of EFI changes
- **EFI_IMAGE_CREATION_GUIDE.md** - Complete guide for creating EFI raw images
- **EFI_QUICK_GUIDE.md** - Quick reference guide for EFI images
- **MANUAL_EFI_BUILD.md** - Step-by-step manual procedure for EFI builds
- **TESTING_SUMMARY.md** - Testing summary and verification steps

### `troubleshooting/`
Troubleshooting guides for common issues:
- **FIX_DOCKER_SOCKET.md** - Solution for Docker socket issues
- **FIX_OOM_ERROR.md** - Solution for Out Of Memory errors
- **OOM_SOLUTION_APPLIED.md** - Applied OOM solution documentation

## Quick Start

### For EFI Boot Support
1. Start with `efi/EFI_QUICK_GUIDE.md` for a quick overview
2. Read `efi/EFI_IMAGE_CREATION_GUIDE.md` for detailed instructions
3. Refer to `efi/MANUAL_EFI_BUILD.md` for manual procedures

### For Troubleshooting
1. Check `troubleshooting/` directory for your specific issue
2. Follow the step-by-step solutions provided

## Contributing

When adding new documentation:
- Place EFI-related docs in `efi/`
- Place troubleshooting guides in `troubleshooting/`
- Use clear, descriptive filenames
- Write in English
- Include code examples where applicable
